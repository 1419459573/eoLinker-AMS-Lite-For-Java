开源版与专业版的区别？

AMS开源版 专为微型团队设计，拥有基础的API管理、测试以及团队协作功能；

AMS专业版 则专注于企业级用户，除了在API管理、基础测试、自动化测试等方面有所强化外，在人员管理方面支持企业级人员架构，允许对用户进行权限分组以及设置详细的操作权限等。
